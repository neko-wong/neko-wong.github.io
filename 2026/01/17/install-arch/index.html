安装Arch Linux和Windows双系统并配置美化Niri桌面

发表于2026-01-17|更新于2026-01-18|Linux
|浏览量:

写在最前面

之前一直用的是Garuda Mocha发行版，但是由于更新chaotic-aur在国内那一言难尽的网速，最终决定折腾一下安装纯净的Arch Linux。
系统效果预览：

准备环境

在windows系统中关闭安全启动和bitlocker！！！

使用磁盘管理压缩出一个新的至少64GB的新分区

下载以下软件

  1. ventoy
  2. 微pe
  3. Arch镜像

准备一个大小约为8GB的U盘，写入ventoy,把微pe和Arch Linux的ISO拷进ventoyU盘里

扩容EFI分区

重启电脑进入微pe环境，打开傲梅分区助手
按照这篇文章将EFI分区扩展至1GB

安装Arch Linux

重启进入arch linux iso

连接无线网络

输入

打开无线网络管理器，输入

按照提示输入密码

磁盘挂载

查看你的efi分区和主分区分区号

我这里就是/dev/nvmeon1p1作为/boot挂载点，/dev/nvme0n1p6作为/挂载点
运行

运行archinstall

设置选项

选择选项为按下空格并回车

除非有经验否则不要乱动其他设置，出现问题后果自负

设置项 设置内容
Locales 保持默认
Mirrors and repositories select regions -> China
Disk configuration Partitioning -> Pre-mounted configuration -> 输入/mnt
Bootloader Bootloader -> Grub
Kernels linux-zen
Hostname 按需求填入
Authentication 第一个是root密码，第二个是用户管理，按需求填入
Profile Type -> Desktop -> niri -> seatd ,返回选择Greeter -> sddm
Applications Bluetooth -> Yes, Audio -> pipewire
Network configuration Copy ISO network configuration to installation
Timezone Asia/Shanghai

最后按下install并确定，完成后重启进入系统

修复Windows启动项

Win+T打开终端

找到最后一行GRUB_DISABLE_OS_PROBER
把True改为False

中国区域优化

配置archlinuxcn和arch4edu

运行dev-sidecar并按照指引配置证书

美化

中文字体

starship

打开~/.config/fish/config.fish 并添加

noctalia shell

设置语言为中文，配色方案下载Catppuccin Lavender,勾选模板-合成器-niri

niri

这里粘贴我的配置，比较潦草

GTK

在nwg-look中选择主题为catppuccin-mocha-green-standard+default

其他应用

按照catppuccin ports的教程进行美化，我的方案是mocha-green

结束

安装你喜欢的应用并开始享受你的Arch Linux吧！
Happy coding!
